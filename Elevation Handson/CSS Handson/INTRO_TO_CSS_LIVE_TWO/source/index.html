Tagname and ID Selectors Demo

This paragraph has an ID of "intro". It will be styled with the following CSS rule: 

      #intro {
        background-color: yellow;
        padding: 10px;
      }

This paragraph does not have an ID or a class. It will be styled with the following CSS rule: 

p {
        font-size: 20px;
    }

More Examples:

This is another paragraph. It will also be styled with the following CSS rule:

      font-size: 20px;
    }

Conclusion

By using tagname selectors and ID selectors, you can target specific elements on your webpage and style them in different ways. This can be useful for creating visually appealing and well-organized webpages.
